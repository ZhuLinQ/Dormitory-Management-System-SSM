# 宿舍管理系统

## 项目介绍

- 本项目是基于SSM（Spring + SpringMVC + MyBatis）框架开发的轻量级宿舍管理系统
- 部署于Tomcat 8.5.24服务器
- 实现宿舍日常管理的数字化、规范化，核心覆盖学生缺寝记录、宿舍调换申请等基础管理功能，操作简单、易于维护。

## 技术栈

### 后端

- 框架：SSM（Spring 5.2.7 + SpringMVC 5 + MyBatis 3.5.5）
- 服务器：Apache Tomcat 8.5.24
- 数据库：MySQL 8.0.21
- 构建工具：Maven 3.6.3
- JDK版本：JDK 1.8

### 前端

- 基础：HTML + CSS + JavaScript
- 模板引擎：JSP
- UI框架：Bootstrap 3
